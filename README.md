Face-Analysis-Pro

Face Analysis Pro is a Streamlit app that analyzes faces in images and videos for age, gender, emotion, and race. It supports real-time analysis, tracks history, and allows downloading processed results.

Features:
- Real-time facial analysis for age, gender, emotion, and race
- Webcam support for live video analysis
- History tracking for past analyses
- Option to download analyzed images

Technologies Used:
- Streamlit: For creating the web interface
- DeepFace: For facial analysis and emotion recognition
- OpenCV: For image and video processing
- Python: Programming language

Installation

Prerequisites:
Make sure you have Python 3.7+ installed on your machine.

Steps to Run Locally:
1. Clone the repository:
   git clone https://github.com/shemaiscard/face-analysis-pro.git
   cd face-analysis-pro
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py

Usage:
- Upload an image or start the webcam for real-time analysis.
- View analysis results such as gender, age, emotion, and race.
- Download the processed image for further use.
